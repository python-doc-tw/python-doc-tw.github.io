關於這些說明文件¶

這些說明文件是透過 Sphinx （一個專為 Python 說明文件所撰寫的文件處理器）將使用 reStructuredText 撰寫的原始檔轉換而成。

如同 Python 自身，透過自願者的努力下產出文件與封裝後自動化執行工具。若想要回報臭蟲，請見 處理bug 頁面，內含相關資訊。我們永遠歡迎新的自願者加入！

致謝：

  • Fred L. Drake, Jr.，原始 Python 文件工具集的創造者以及一大部份內容的作者。
  • 創造 reStructuredText 和 Docutils 工具組的 Docutils 專案；
  • Fredrik Lundh 先生，Sphinx 從他的 Alternative Python Reference 計劃中獲得許多的好主意。

Python 文件的貢獻者們¶

許多人都曾為 Python 這門語言、Python 標準函式庫和 Python 說明文件貢獻過。Python 所發佈的原始碼中含有部份貢獻者的清單，請見 Misc/ACKS 。

正因為 Python 社群的撰寫與貢獻才有這份這麼棒的說明文件 – 感謝所有貢獻過的人們！
